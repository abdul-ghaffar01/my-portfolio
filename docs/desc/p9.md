## Why I Built This

I wanted a **full-featured e-learning platform** to showcase my skills in backend, frontend, database design, and deployment.  
This project aims to provide a seamless learning environment with user authentication, course creation, video lessons, quizzes, student progress tracking, and an admin/instructor dashboard.  
By building this platform I also wanted to practice **real-world architecture patterns, clean API design, responsive UI, and deployment workflows**.

Since the project is **still ongoing**, I plan to add features like notifications, analytics, interactive assignments, and possibly real-time chat or discussion threads for students in future updates.
